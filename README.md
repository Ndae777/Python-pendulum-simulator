# Multi-Pendulum Simulator

## 📌 Description
This project is a physics-based simulation of multiple pendulums, built using Python. It combines motion physics, GUI elements, and real-time animation to visualize pendulum behavior.

The simulation allows users to control the number of pendulums and their properties, creating a dynamic and interactive experience.

## 🚀 Features
- Multi-pendulum simulation with real-time motion
- Physics-based movement using gravity and damping
- Randomized pendulum lengths and colors
- Dynamic GUI using Tkinter
- Animated visualization using the Turtle module
- Clock-style motion indicators

## 🛠️ Technologies Used
- Python
- Turtle Graphics
- Tkinter (GUI)
- Math (physics calculations)
- Random module

## ▶️ How to Run
1. Install Python (if not already installed)
2. Run the script:
   python main.py

3. A GUI window will open where you can:
   - Select number of pendulums
   - Set minimum and maximum length
   - Start the simulation
